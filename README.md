##所用工具(js库,css库)
* js：[zepto.js](https://github.com/madrobby/zepto)
* css3: [animate.css](http://daneden.github.io/animate.css/)

##功能
* 图片列表4张图片一排分布
* 触摸小图出现大图
* 实现大图左滑、右滑翻页
